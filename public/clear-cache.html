auxOS Cache Cleaner

What this does:
This will clear your auxOS file system cache so you can see the updated file names (auxe_intel.md, auxe_services.md) in the Finder.

Manual Steps:

  1. Click "Clear File System Cache" above
  2. Go back to auxOS and refresh the page
  3. Open Finder - you should now see the new file names

Alternative Method:

Open auxOS in an incognito/private window to see the changes without clearing cache.
